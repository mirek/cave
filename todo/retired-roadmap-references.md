# Replace retired roadmap references

## Problem

Help and package descriptions still cite roadmap item numbers, requiring readers to consult a historical appendix.

## Direction

Replace user-facing roadmap references with specification sections or capability names while retaining the retired roadmap for history.

## Done when

- User-facing help and package metadata contain no unexplained roadmap numbers.
- Historical source comments are updated where the spec is a clearer anchor.
- Links resolve to current documentation.
