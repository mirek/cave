Model parenthood as relations, not attributes: declare `PARENT-OF IS verb`
and `PARENT-OF REVERSE CHILD-OF` once, then write claims like
`helena PARENT-OF jan`. Record disputed values (e.g. a contested birth
year) as separate claims, each with its own `@src:` context and
confidence. Prefer short entity names: `jan`, `helena`, `helena/father`,
`me` for the note's author.
